- Умный калькулятор чаевых

Простая консольная утилита на Python для расчета чаевых и разделения счета в компании.

-- Функции

*   Расчет общей суммы к оплате с учетом чаевых.
*   Расчет суммы чаевых.
*   Разделение итоговой суммы на любое количество человек.

-- Использование

1.  Запустите файл `src/tip_calculator.py`.
2.  Следуйте инструкциям в консоли.
3.  Введите сумму счета, процент чаевых и количество человек.

-- Пример

~~~
Введите сумму счета: 2500
Введите процент чаевых, который хотите оставить (например, 15): 10
На сколько человек нужно разделить счет? (Нажмите Enter, если только на себя): 4

Результаты расчета:
Общая сумма к оплате: $2750.0
Сумма чаевых: $250.0
С каждого человека: $687.5 (включая чаевые $62.5)
~~~
